# PruebaTecnica

Pasos para correr el backend:
- Ubicarse sobre la carpeta "api/"
- Ejecutar el comando en la terminal "node server.js"
- Si es satisfactorio, debe salir un mensaje "Servidor escuchando en http://localhost:3000"
- Endpoints disponibles: 
1. GET http://localhost:3000/productos
2. POST http://localhost:3000/productos
3. DELETE http://localhost:3000/productos/{id}
